Dessert Recipe Collection

A visually interactive web project featuring step-by-step guides for specialty desserts. This project uses a combination of HTML, CSS, and JavaScript to create a themed browsing experience.

🍰 Featured Recipes

Lemon Blueberry Cheesecake Soft Cookies: A multi-stage recipe featuring a frozen cream cheese filling, homemade blueberry jam, and a lemon-zest cookie dough.
Lemon Cake: A classic dessert layered with custom lemon syrup and a smooth lemon icing finish.

✨ Interactive Features

Themed Animations: Includes custom JavaScript "rain" effects (🍋 and 🍪) that trigger when navigating between recipes.
Pastel UI: Designed with a color-coded interface—pastel brown for cookies and pastel yellow for cake—to match the aesthetic of the desserts.
Air Fryer Optimized: The cookie recipe includes specific settings for air fryer baking at 160°C.

🛠️ Technical Details

Languages: HTML5, CSS3, and JavaScript.
Styling: Utilizes specific font-face declarations (Times New Roman) and transparency-based content boxes for high readability against image backgrounds.
Assets: This project requires `blueberry.png` and `lemoncake.png` for full visual effect.

🚀 How to Use

1.  Upload all `.html` and `.png` files to your web server or GitHub repository.
2.  Open `Cheesecake.html` or `Lemon.html` in any modern web browser.
3.  Use the sidebar menu to toggle between recipes and enjoy the transition animations.
